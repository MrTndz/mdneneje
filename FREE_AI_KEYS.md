# 🆓 Как получить БЕСПЛАТНЫЕ AI ключи

## ⭐ DeepSeek (РЕКОМЕНДУЕТСЯ - самый лучший бесплатный)

**Почему DeepSeek:**
- Полностью бесплатный
- Очень быстрый
- Качество как у GPT-4
- Огромный бесплатный лимит
- API совместим с OpenAI

**Как получить:**

1. Откройте: https://platform.deepseek.com
2. Нажмите "Sign Up" (регистрация)
3. Зарегистрируйтесь через email или Google
4. После входа перейдите в API Keys
5. Нажмите "Create API Key"
6. **Скопируйте ключ** (начинается с `sk-`)
7. Вставьте в `index.js` строка 27:
   ```javascript
   const DEEPSEEK_KEY = "sk-ваш_ключ_здесь";
   ```

**Бесплатный лимит:**
- 10 миллионов токенов в месяц
- Этого хватит на ~50,000 сообщений!

---

## 🤗 Hugging Face (альтернатива)

**Почему Hugging Face:**
- Полностью бесплатный
- Огромный выбор моделей
- Без ограничений на количество запросов

**Как получить:**

1. Откройте: https://huggingface.co/join
2. Зарегистрируйтесь
3. После входа перейдите: https://huggingface.co/settings/tokens
4. Нажмите "New token"
5. Название: "MerAI Bot"
6. Тип: "Read"
7. Нажмите "Generate"
8. **Скопируйте ключ** (начинается с `hf_`)
9. Вставьте в `index.js` строка 28:
   ```javascript
   const HF_KEY = "hf_ваш_ключ_здесь";
   ```

**Лимит:**
- Бесплатно без ограничений
- Может быть чуть медленнее DeepSeek

---

## ⚡ Groq (платный, но с бесплатным тарифом)

**Как получить:**

1. Откройте: https://console.groq.com
2. Зарегистрируйтесь
3. Перейдите в API Keys
4. Создайте новый ключ
5. Вставьте в `index.js` строка 29

**Бесплатный лимит:**
- 14,400 запросов в день
- Очень быстрый

---

## 🔮 Google Gemini (платный с бесплатным тарифом)

**Как получить:**

1. Откройте: https://aistudio.google.com/apikey
2. Войдите через Google
3. Нажмите "Create API Key"
4. Выберите проект или создайте новый
5. Скопируйте ключ
6. Вставьте в `index.js` строка 30

**Бесплатный лимит:**
- 60 запросов в минуту
- 1,500 запросов в день

---

## 💡 Рекомендация

**Для начала используйте:**
1. **DeepSeek** - лучший бесплатный вариант
2. **Hugging Face** - как запасной вариант

**Не нужны:**
- Groq (если ваш ключ уже заблокирован)
- Gemini (если есть DeepSeek)

---

## 🚀 Установка ключей

После получения ключей откройте `index.js` и найдите строки 27-30:

```javascript
// БЕСПЛАТНЫЕ AI ПРОВАЙДЕРЫ
const DEEPSEEK_KEY  = "sk-ваш_ключ_deepseek";  // ⬅️ СЮДА
const HF_KEY        = "hf_ваш_ключ_huggingface";  // ⬅️ СЮДА
const GROQ_KEY      = "";  // Можно оставить пустым
const GEMINI_KEY    = "";  // Можно оставить пустым
```

Сохраните файл и перезапустите бота:
```bash
node index.js
```

Вы увидите:
```
✅ AI провайдеры: DeepSeek (бесплатный), HuggingFace (бесплатный)
```

Готово! ✅
