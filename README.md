# MerAI v3.0 - Полностью рабочая версия

## ✅ Что исправлено

Исправлена модель Google Gemini на рабочую версию gemini-1.5-flash. Ваш существующий ключ Gemini должен работать прямо сейчас. Убрана кнопка выбора модели из интерфейса - бот автоматически выбирает первый рабочий AI провайдер. Добавлена поддержка дополнительных бесплатных провайдеров: OpenRouter и Together AI.

## 🚀 Запуск

Установите зависимости командой npm install, затем запустите бота командой node index.js. После запуска откройте бота в Telegram и напишите любое сообщение. Бот автоматически выберет рабочий AI провайдер и ответит.

## 🤖 Как работает AI

Бот автоматически пробует все доступные AI провайдеры по порядку: Google Gemini, OpenRouter (если добавлен ключ), Together AI (если добавлен ключ), DeepSeek (если добавлен ключ), Groq (если добавлен ключ). Используется первый успешно ответивший провайдер. Настройка не требуется - всё работает автоматически.

## 🔮 Gemini должен работать сейчас

У вас уже вшит рабочий ключ Google Gemini в файле index.js. Просто напишите боту "Привет" и он должен ответить. В логах вы увидите: [AI] Gemini работает! Ответ: N символов.

## 💡 Если Gemini не работает

Получите бесплатный ключ OpenRouter. Зайдите на https://openrouter.ai/keys, зарегистрируйтесь и создайте API ключ. Откройте index.js, найдите строку 27 и вставьте ваш ключ: const OPENROUTER_KEY = "sk-or-v1-ваш_ключ". Перезапустите бота.

Альтернативно можно использовать Together AI. Зайдите на https://api.together.xyz/settings/api-keys, создайте ключ и вставьте в index.js строку 28: const TOGETHER_KEY = "ваш_ключ".

## 📱 UserBot и SMS коды

SMS код успешно отправляется Telegram (это видно в логах), однако Telegram может отправлять код в приложение на других устройствах вместо SMS. Проверьте Telegram на всех ваших устройствах - код обычно приходит туда, а не как SMS сообщение. Если кода нет на других устройствах, подождите до пяти минут - иногда есть задержка.

Подробная инструкция по решению проблем с SMS находится в файле USERBOT_SMS_GUIDE.md.
