## 📱 UserBot - Решение проблем с SMS

**Код отправляется, но не приходит?**

**Причина**: Telegram отправляет код в другое устройство или с задержкой

**Решение**:
1. Проверьте Telegram на ВСЕХ устройствах
2. Код может прийти в приложение, а не SMS
3. Подождите 5 минут
4. Закройте Telegram на всех устройствах кроме одного

📖 **Подробная инструкция**: `USERBOT_SMS_GUIDE.md`

## 🔧 Решение ошибок

### AI не работает (403, 404)
**Проблема**: Старые/неверные ключи

**Решение**:
1. Получите БЕСПЛАТНЫЙ ключ DeepSeek: https://platform.deepseek.com
2. Вставьте в `index.js` строка 27:
   ```javascript
   const DEEPSEEK_KEY = "sk-ваш_ключ_здесь";
   ```
3. Перезапустите бота

### UserBot: SMS не приходит
**Проблема**: Код отправляется в Telegram, а не SMS

**Решение**: См. `USERBOT_SMS_GUIDE.md`
