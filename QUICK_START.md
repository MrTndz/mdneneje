# БЫСТРЫЙ СТАРТ - 2 шага

## Шаг 1: Запустите

```bash
npm install
node index.js
```

## Шаг 2: Тестируйте

Откройте бота в Telegram и напишите:
```
Привет
```

## Что произойдёт

Бот автоматически:
1. Попробует Google Gemini (ключ уже есть)
2. Если не сработает - попробует другие провайдеры
3. Ответит первым рабочим AI

## В логах увидите

```
[AI] 🤖 Ищу рабочий AI провайдер...
[AI] 🔮 Пробую Google Gemini...
[AI] ✅ Gemini работает! Ответ: 156 символов
```

## Если Gemini не работает

Получите бесплатный ключ OpenRouter:

1. https://openrouter.ai/keys
2. Создайте ключ
3. Вставьте в `index.js` строка 27
4. Перезапустите

## Готово!

AI выбирается автоматически. Никаких настроек не нужно.
